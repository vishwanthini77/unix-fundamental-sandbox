# Session 4 Solutions: Package Management (Brief Guide)

## Ubuntu/Debian (apt)
```bash
sudo apt update
sudo apt install nginx
sudo apt remove nginx
```

## Amazon Linux/RHEL (yum)
```bash
sudo yum install nginx
sudo yum remove nginx
```

## Directory Usage
- `/opt/myapp` - Application code
- `/etc/myapp` - Configuration files
- `/var/log/myapp` - Log files
- `/var/lib/myapp` - Application data
